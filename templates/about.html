About This Project

What is This?

An AI-powered web application that recognizes handwritten digits from images using state-of-the-art machine learning algorithms.

Technology Stack

  • Python 3.10+
  • Django 4.2+
  • Scikit-learn
  • Random Forest & KNN
  • HTML5, CSS3, JS
  • GSAP Animations

Models Used

  • Random Forest (Best)
  • K-Nearest Neighbors
  • Decision Tree
  • Logistic Regression
  • AdaBoost

Performance

  • Accuracy: 97.8%
  • Response: <100ms
  • 50,000+ images trained
  • 5 ML models evaluated

Future Roadmap

  • Real-time drawing recognition
  • Multi-digit recognition
  • API for developers
  • Mobile app version
  • More ML models (CNN, Transformers)
